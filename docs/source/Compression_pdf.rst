France Compression de PDF
=========================

France Compression de PDF est un logiciel pratique qui permet de **réduire la taille des fichiers PDF**. 

Le programme utilise deux méthodes principales pour compresser vos documents :

- **Réduction de la qualité des images** contenues dans le PDF.
- **Compression ZLIB** pour compresser les données internes du PDF.

Instructions d'utilisation
--------------------------

1. Lancez le programme en double-cliquant sur l’exécutable ou en exécutant le script Python.
2. Choisissez le fichier PDF à compresser.
3. Ajustez les paramètres de **qualité d’image** et de **niveau de compression** selon vos besoins.
4. Sauvegardez le PDF compressé.
