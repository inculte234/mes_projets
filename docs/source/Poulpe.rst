Poulpe, l'assistant de veille sur les marchés publics
==================

Bienvenue sur la page de téléchargement de **Poulpe**, un petit assistant de veille pour les marchés publics. Cette page explique comment obtenir la dernière version du logiciel,
comment l’installer et comment vérifier vos licences.

.. contents::
   :local:
   :depth: 2


Présentation
------------

**Poulpe** est un outil qui simplifie la recherche et l'analyse des marchés publics.
Il télécharge automatiquement les DCE, les décompresse, analyse les documents et extrait
les informations pertinentes pour accélérer le sourcing.

Le logiciel fonctionne sous Windows et ne nécessite aucune installation particulière :
il suffit de télécharger l’archive et de lancer l’exécutable.


Téléchargement
--------------

Vous pouvez télécharger la dernière version stable de Poulpe ci-dessous :

**Dernière version stable :**  
`poulpe-latest.zip <https://github.com/inculte234/mes_projets/releases/download/Poulpe/poulpe.0.1.zip>`__


L'archive contient :

* ``ouverture_poulpe.exe`` – le lanceur principal
* ``poulpe_recherche_licensed.exe`` – le module de recherche
* ``poulpe_analyse_licensed.exe`` – le module d'analyse


Installation
------------

Aucune installation n'est requise.

1. Téléchargez le fichier ``poulpe.zip`` depuis le lien ci-dessus.
2. Décompressez le fichier dans le dossier de votre choix.
3. Double-cliquez sur ``ouverture_poulpe.exe``.

Licence
-----------

Poulpe fonctionne avec une licence. Vous pouvez demander cette licence à un administrateur.
